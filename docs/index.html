🚀 Quick Start

Single Player

  1. Run the game
  2. Press ENTER at main menu
  3. Select difficulty (↑/↓)
  4. Play! (W/S or ↑/↓)

Multiplayer

  1. Press O at main menu
  2. Login or Register (R)
  3. Pick a persistent room or enter ID
  4. Wait for opponent or share your ID
