# Neon Pong · Cyberpunk Edition

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Java Version](https://img.shields.io/badge/Java-11%2B-blue)](https://www.oracle.com/java/technologies/javase-downloads.html)
[![Version](https://img.shields.io/badge/version-2.2β1-purple)](https://github.com/github-deewhy/pong-neon)

A visually-stylized, multiplayer-enabled Pong game with a cyberpunk aesthetic. Features online multiplayer via PocketBase, procedural sound engine, and neon visual effects.

![Neon Pong](docs/screenshot.png)

## ✨ Features

- **Single-player** with 4 AI difficulty levels (Rookie, Veteran, Elite, Legend)
- **Online multiplayer** with persistent rooms and account system
- **Procedural sound engine** – no external files needed
- **Cyberpunk visuals** – neon gradients, particle effects, screen shake
- **Account management** – register, login, change email/password
- **Cross-platform** – runs on any system with Java 11+

## 🚀 Quick Start

### Prerequisites
- Java 11 or higher

### Running the game
```bash
# Compile
javac beta-test/multiplayer/PongGame.java

# Run
java -cp beta-test/multiplayer PongGame
